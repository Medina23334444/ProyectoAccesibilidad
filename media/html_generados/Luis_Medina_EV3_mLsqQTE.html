A. Caso de Estudio: Sistema de Puntos de Venta (POS) para ZERIMAR.
Contexto de la Empresa: ZERIMAR es una cadena de tiendas minoristas en expansión que
actualmente opera con procesos manuales y hojas de cálculo para controlar sus ventas e
inventarios. La gerencia ha decidido dar el salto a la transformación digital mediante el desarrollo
de un Sistema de Puntos de Venta (POS) centralizado.
Alcance del Proyecto: El sistema debe permitir el registro de ventas en tiempo real, la gestión
de inventarios (stock de productos), un módulo de fidelización de clientes (puntos) y la generación
de reportes de cierre de caja. Además, el sistema debe integrarse con un software contable
externo y permitir la consulta de stock entre sucursales.
Desafíos y Factores Críticos:
1. Infraestructura: Varias sucursales se encuentran en zonas con conectividad a internet
intermitente.
2. Seguridad: El módulo de fidelización maneja "puntos" con valor monetario, lo que atrae
riesgos de fraude.
3. Cumplimiento Legal: El sistema debe cumplir estrictamente con las normativas locales de
facturación electrónica y cambios de impuestos.
4. Integración: La conexión con el sistema contable externo es mediante una API de terceros
que no ha sido probada previamente por el equipo de desarrollo.
5. Usuarios: El personal de cajas tiene una alta rotación y niveles variables de alfabetización
digital.
